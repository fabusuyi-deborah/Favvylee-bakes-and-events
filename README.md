# Favvylee Bakes and Events

A responsive baking, catering services and event website that showcases custom cakes, fresh baked goods, catering services and event planning built with frontend technologies and best practices.

Live Site : [favvylee bakes and events](https://favvylee-bakes-and-events.vercel.app/)

## Preview
![Preview](/public/website-screen.png)

## Features

- Responsive, mobile-first design
- Services and gallery showcase
- Direct WhatsApp ordering with pre-filled message
- Clean and modern UI

## Tech Stack
- Next.js
- React
- Tailwind CSS
- TypeScript
- Deployed on Vercel

## Getting Started
npm install
npm run dev
